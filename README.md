# 💰 SPENDWISE — Expense Tracker UI/UX Design

This project is part of my **UI/UX Design Internship — Task 3**, where I created a complete **Design System** and **mobile application UI** for an expense tracking app called SPENDWISE.

## 🚀 Project Overview
SPENDWISE is a mobile expense tracking application that helps users manage daily expenses, track spending categories, and analyze financial data.

The main focus of this task was to build a **scalable design system** and implement consistent UI screens using Figma.
